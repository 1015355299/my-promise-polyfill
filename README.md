# my-promise-polyfill

es5的promise实现

## 项目中引入

1. 安装 `npm i my-promise-polyfill --save`


2. 引入

```js
// 自动进行polyfill
require('my-promise-polyfill')
```

## 二次开发步骤

1. `git clone git@github.com:1015355299/my-promise-polyfill.git` 克隆项目

2. `cd my-promise-polyfill` 进入项目

3. `npm install` 安装依赖

4. `npm run test` 测试

5. `npm run build` 开发后重新打包
